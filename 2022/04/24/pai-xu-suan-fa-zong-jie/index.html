排序算法的稳定性

稳定的排序：

  • 冒泡排序
  • 插入排序
  • 归并排序
  • 计数排序

上述基于比较的排序算法在遇到相同的值时不做交换，这样保证了稳定性；

不稳定排序：

  • 快速排序
  • 堆排序

冒泡排序

如果相邻的两个元素处于错误的顺序，那么交换这两个元素，如此重复。

void bubbleSort(vector<int>& nums) {
    int n = nums.size();
    for (int i = 0; i < n - 1; i++) {   // 外层循环，n - 1次，每次将最大值沉底，下次排序的范围-1
        for (int j = 0; j < n - i - 1; j++) {   // 内层循环，在排序范围内比较相邻的两个元素是否有序
            if (nums[j] > nums[j + 1]) {
                swap(nums[j], nums[j + 1]);
            }
        }
    }
}

插入排序

对元素 ，在前面的 个位置中找到它应该插入的位置。向前寻找位置的过程中向后移动元素，腾出空间。插入排序是一种稳定的排序算法。

void insertSort(vector<int>& nums) {
    int n = nums.size();
    for (int i = 1; i < n; i++) {
        int key = nums[i];
        int j = i - 1;
        while (j >= 0 && nums[j] > key) {
            nums[j + 1] = nums[j];
            j--;
        }
        nums[j + 1] = key;
    }
}

堆排序

堆排序是一种基于比较的排序算法，其底层数据结构为二叉堆。堆排序和选择排序相似，选择排序中我们每次选择最小值放在数组的头部，然后对剩余的元素重复这样的过程。堆排序也是如此，以大顶堆为例，我们每次选择堆顶元素，将其与数组的末尾元素交换，这样最大值就放在了数组的尾部，此时堆结构被破坏，需要重新调整使其满足堆的结构。然后对剩余的前 n - 1 个元素重复这样的过程。

为什么堆使用数组来表示？

因为堆的数据结构是完全二叉树，很容易用数组来表示，并且空间效率很高。如果父节点的下标是 ，那么它的孩子节点下标分别是 和 。(下标从0开始)

堆排序的过程：

  1. 根据输入数据建立一个大顶堆；
  2. 建好堆以后，堆顶元素即为最大值。将堆顶元素与数组末尾元素互换，然后将堆的 ，然后调整堆的结构，也就是对堆顶元素进行 ；
  3. 重复 step2 直到堆的 。

C++代码实现

// 把以下标i为根的子树堆化，n是堆的大小
void heapify(vector<int> nums, int n, int i) {
    int largest = i;	// 初始化largest为根的下标
    int left = i * 2 + 1;	// 左孩子
    int right = i * 2 + 2;	// 右孩子
    
    //如果左孩子在堆的范围内，并且左孩子比根节点大，更新largest为左孩子的下标
    if (left < n && nums[left] > nums[largest]) {
		largest = left;
    }
    //如果右孩子在堆的范围内，并且右孩子比根节点大，更新largest为右孩子的下标
    if (right < n && nums[right] > nums[largest]) {
        largest = right;
    }
    // 如果根节点已经不是最大值，交换根节点与最大值，继续调整
    if (largest != i) {
        swap(nums[i], nums[largest]);
        heapify(nums, n, largest);
    }
}

void heapSort(vector<int>& nums) {
    int n = nums.size();
    // 建堆
    for (int i = n / 2 - 1; i >= 0; i--) {
        heapify(nums, n, i);
    }
    // 从最后一个元素开始与堆顶元素交换，同时i也是当前堆的大小
    for (int i = n - 1; i > 0; i--) {
        swap(nums[0], nums[i]);
        heapify(nums, i, 0);
    }
}

C++优先队列

priority_queue<int, vector<int>, less<int>> q;

参数列表：

  • 第一个参数 int ：表示优先队列存储的数据类型；
  • 第二个参数 vector<int>：表示优先队列的底层使用什么容器来存储数据，一般使用 vector ；
  • 第三个参数 less<int>：比较类型，less 表示是大顶堆，great 表示是小顶堆；

自定义数据类型时需要重载运算符 < 或者重写一个比较类型：

struct Node{
	int num;
    ListNode* root;
    // 重载运算符：
    bool operator < (const Node& a) const {
        return num < a.num;		// 大顶堆
    }
}
// 创建一个优先队列，大顶堆
priority_queue<Node, vector<Node>, less<Node>> q;

// 重写比较类型
struct cmp {
    bool operator() (Node a, Node b) {
        return a.num < b.num;	// 大顶堆
    } 
};
// 创建一个优先队列，大顶堆
priority_queue<Node, vector<Node>, cmp> q;
